# Reports

This directory contains the project documentation for the **Posture Corrector using ATmega328P and MPU6050**.

The reports provide detailed information about the project's objectives, design methodology, implementation, testing, and outcomes. These documents serve as a comprehensive reference for understanding the development process and technical aspects of the project.

---

## Contents

### Project Report
The complete project report, including the problem statement, objectives, literature review, system design, implementation, testing, results, and conclusion.

### Presentation
Presentation slides prepared for project reviews, demonstrations, or evaluations.


## Purpose

The documents in this directory provide a detailed record of the project's development and complement the hardware and firmware resources available in this repository.
