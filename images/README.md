# Images

This directory contains photographs and visual assets related to the **Posture Corrector using ATmega328P and MPU6050** project.

The images provide a visual overview of the hardware development, PCB fabrication, assembly process, and the final working prototype.

---

## Contents

### Final Prototype
Photographs of the completed wearable posture corrector.

### PCB
Images of the fabricated printed circuit board, including top and bottom views.

### Hardware Assembly
Photographs of the assembled electronic components and wiring.


### PCB Layout and 3D View
Rendered images of the PCB layout and 3D model generated during the design phase.

---

## Purpose

The images included in this directory document the development process and provide visual reference for the hardware implementation, making it easier to understand the overall design and construction of the project.
