# Results

This directory contains the testing results and performance evaluation of the **Posture Corrector using ATmega328P and MPU6050**.

The results demonstrate the functionality of the system, including posture detection accuracy, alert response, and overall hardware performance under different testing conditions.

---

## Contents

### Functional Testing
Results obtained during system testing to verify correct operation of the hardware and firmware.

### Posture Detection
Observations of the system's ability to distinguish between correct and incorrect posture based on sensor data.

### Alert Mechanism
Verification of the buzzer or vibration motor response when poor posture is maintained beyond the predefined threshold.

### Calibration
Calibration data and test observations used to improve sensor accuracy and reliability.

### Demonstration
Screenshots, photographs, or recorded outputs showcasing the successful operation of the posture correction system.

---

## Summary

The implemented system successfully monitors body posture in real time using the MPU6050 IMU and ATmega328P microcontroller. The device detects prolonged poor posture and provides immediate feedback through the alert mechanism, demonstrating the feasibility of a compact and low-cost wearable posture monitoring solution.
