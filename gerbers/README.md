# Gerber Files

This directory contains the Gerber files generated from the final PCB design of the **Posture Corrector** project.

Gerber files are the standard manufacturing files used by PCB fabrication services to produce the printed circuit board. They include all the information required to manufacture the PCB, such as copper layers, solder mask, silkscreen, board outline, and drill data.

## Contents

- Copper layer files
- Solder mask files
- Silkscreen files
- Board outline
- NC drill files

## Usage

The files in this directory can be compressed into a ZIP archive and uploaded directly to PCB manufacturers such as **JLCPCB**, **PCBWay**, or **Seeed Fusion** for fabrication.

## Notes

- Generated from the final verified PCB layout.
- Ensure all Gerber files and drill files are included before submitting for fabrication.
- Perform a Gerber viewer check to verify the design before manufacturing.
